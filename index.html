The 75 Hard Challenge

For the next 75 days I am:

  1. Dieting.

    • Stay below 100% of the recommended daily value for added sugars.

      • Natural sugars found in things like fruit are okay.

  2. Working out for 45 minutes, twice a day.

    • One workout must be outside.

    • 3 pull-ups whenever going downstairs.

      • This one doesn't count towards workouts.

  3. Drinking one gallon of water.

    • No flavorings, or substitutions with other liquids.

  4. Coding something everyday.

    • Github contributions must be green for 75 days straight.

  5. Taking a progress picture.
